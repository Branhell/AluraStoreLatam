# Alura Store – Análisis de Desempeño por Tienda

## Propósito del Proyecto

Este proyecto analiza el rendimiento de cuatro tiendas de la cadena Alura Store, utilizando Python y Pandas para generar insights sobre ventas, satisfacción del cliente, productos más vendidos y distribución geográfica.

## Tecnologías Utilizadas

- Python 3
- Pandas
- Matplotlib
- Folium (opcional para mapas interactivos)
- JupyterLab
